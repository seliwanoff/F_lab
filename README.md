FormatLab is a Chrome extension that combines a **JSON Formatter**, **syntax-highlighted JSON viewer**, and **global network request monitor** â€” all in one developer toolkit.  

Built with **React**, **TypeScript**, and **TailwindCSS**, it makes debugging and inspecting JSON and network requests effortless.

---

## Features

- **Format & Minify JSON** instantly  
- **Syntax-highlighted JSON** display with `react-syntax-highlighter`  
- **Dark & Light modes** for comfortable viewing  
- **Global network request capture** across all tabs via background service worker  
- **Copy JSON to clipboard**  
- **Request history** saved in `chrome.storage`
